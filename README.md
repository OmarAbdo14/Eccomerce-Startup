# Eccomerce

## Commands
* setup
```
$ composer i
$ npm i
$ php artisan key:generate
$ php artisan migrate
```
* Run the serve
update the hostname and the portname in the resources/js/App/APIs.js too
```
$ php artisan serve --host=<hostname> --port=<portname>
```

* Run the npm for UI updates
```
$ npm run dev
$ npm run watch
```
